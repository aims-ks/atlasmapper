== Mercurial help ==Check-out the project:$ hg clone https://code.google.com/p/atlasmapper/Commit update to the local server$ hg commitGet update from the live server$ hg pull https://code.google.com/p/atlasmapper/Commit update to the live server$ hg push https://code.google.com/p/atlasmapper/== Building the AtlasMapper from Source ==This section will eventually describe the building of the source to produce an executable WAR file.
