== Mercurial help ==Check-out the project:$ hg clone https://code.google.com/p/atlasmapper/$ cd atlasmapperCommit update to the local server$ hg commitGet update from the live server$ hg pull https://code.google.com/p/atlasmapper/Commit update to the live server$ hg push https://code.google.com/p/atlasmapper/user: Your google ID, the one used to log into google (without the @gmail.com)password: Can by found at the URL https://code.google.com/hosting/settings== Building the AtlasMapper from Source ==This section will eventually describe the building of the source to produce an executable WAR file.
